CLI Directory Scanner

Простой CLI-скрипт на PHP для рекурсивного сканирования директорий.

Поддерживает:
	•	вывод дерева файлов
	•	ограничение глубины
	•	фильтрацию по расширению

Скрипт предназначен для запуска из командной строки.

⸻

Запуск

Базовый запуск (сканируется / без ограничений):

php scan.php


⸻

Примеры запуска

1. Ограничение глубины

Сканировать корень / только на 2 уровня:

php scan.php --depth=2


⸻

2. Фильтрация по расширению

Показывать только .php файлы:

php scan.php --ext=php

Допустимы оба варианта:

php scan.php --ext=php
php scan.php --ext=.php


⸻

3. Фильтр + глубина

Только PHP-файлы, не глубже 3 уровней:

php scan.php --ext=php --depth=3


⸻

4. Своя директория

Сканировать конкретный путь:

php scan.php --dir=/var/www --depth=2


⸻

5. Полная комбинация аргументов

php scan.php --dir=/var/www --ext=php --depth=2


⸻

Поддерживаемые аргументы

Аргумент	Описание
--dir	Корневая директория (по умолчанию /)
--ext	Фильтр по расширению файла
--depth	Максимальная глубина рекурсии


⸻

Примечания
	•	При сканировании / часть директорий может быть недоступна — это нормально
	•	Недоступные директории помечаются как [NO ACCESS]
	•	Рекомендуется всегда использовать --depth при работе с /
